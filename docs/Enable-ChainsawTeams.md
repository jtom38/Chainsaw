# Enable-ChainsawTeams

## SYNOPSIS

This lets you enable chainsaw to send to Microsoft Teams Webhooks.

## SYNTAX

```PowerShell
Enable-ChainsawTeams [[-URI] <String>] [[-Levels] <String[]>] [[-MessageTitle] <String>] [-ScopeGlobal]
 [[-JsonConfig] <String>]
```

## DESCRIPTION

This is a wrapper for PSTeams.
